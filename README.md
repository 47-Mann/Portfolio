# Sahil Mann Portfolio

A personal portfolio website built with React and Vite to showcase projects, technologies, and professional profile details.

## Features

- Responsive portfolio layout
- Animated hero section with typing effect
- Project showcase section
- Technologies/skills section with icons
- Dark mode toggle
- Modern React component-based structure

## Tech Stack

- React
- Vite
- CSS3
- React Icons
- React Type Animation

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/47-Mann/Portfolio.git
cd Portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```
